    草稿

    這樣會在 /source 資料夾下新增一篇草稿

    如果要將草稿轉成正式文章有兩種方式：

    1. 手動搬到 /_post 目錄底下
    2. 使用 hexo publish draft <title>

    另外，由於草稿在本地 hexo server 時也看不到 XD 如果想要看得到的話也有兩個方法：

    1. hexo server 時多加上 --draft 參數：
    2. 編輯 _config.yml 檔案，將 render_drafts 設定成 true
